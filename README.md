# mxcl/made

Homebrew formulae for projects maintained or patched by Max Howell.

```sh
brew tap mxcl/made
```

## Projects

| Package | Project | Description |
| --- | --- | --- |
| `cig` | [mxcl/cig](https://github.com/mxcl/cig) | Checks Git repositories for uncommitted or unpushed changes. This fork fixes handling of ignored nested repositories. |
| `clawlicious` | [mxcl/clawlicious](https://github.com/mxcl/clawlicious) | Reads and files bookmarks with Codex. |
| `pkg⋅mgr²` | [mxcl/package-manager-manager](https://github.com/mxcl/package-manager-manager) | Inventories and manages packages across package managers. |
| `swift-sh` | [mxcl/swift-sh](https://github.com/mxcl/swift-sh) | Swift scripting with zero-configuration dependency imports. |
| `git-cryobank` | [mxcl/git-cryobank](https://github.com/mxcl/git-cryobank) | Temporary archiving for experimental projects. |

Install a package with `brew install mxcl/made/<package>`.
